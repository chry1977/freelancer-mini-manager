# Freelancer Mini Manager

A simple offline tool to track clients, jobs, and payments.

This tool runs directly in your browser.
No accounts, no subscriptions, no cloud.
All data stays locally on your device and can be exported to CSV.

## Features
- Client, job, and payment tracking
- Works fully offline
- Local storage (data stays on your device)
- CSV export
- English and Italian language support
- Minimal and lightweight

## How to use
1. Download the files
2. Open `index.html` in your browser
3. Do not use incognito mode
4. Start tracking your work
